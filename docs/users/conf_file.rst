##################
Configuration File
##################

The config file acts as the sole input for TTiP and has many customisable options.

These options can be seen below. After an example config which describes the
options, some information is given on how to input complex functions which may be
required for your problem.

To create a local copy of this to edit, the ``gen_conf`` option can be used::

   ttip --gen_conf
